Box-Packing Tester

A project of North American Web Company

What is this?

This is a tool for testing the Box-Packer API and the box-packing algorithm implementation it exposes. To use the tool, enter dimensions and (optionally) a description for each of your items and boxes, and the tool will tell you whether all the items fit into all the boxes.

Fork this project, the Box Packer API, or the Box Packer Composer package on GitHub.
