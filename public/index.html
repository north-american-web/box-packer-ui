Let's Pack Some Boxes

This is a simple tool for testing whether items will fit into a given set of containers (boxes). Enter dimensions and (optionally) a description for each of your items and boxes, and the tool will tell you whether all the items fit into all the boxes.

The “bin packing problem” applies here. While this tool doesn't guarantee an optimal solution (i.e., for some combinations of items and boxes, it will falsely claim the items don't fit), but it is fast and probably good-enough for most purposes.

How to use

An item or box's specs should be entered in the following format: width, length, height, description. Note that the values should be comma separated and the order of the dimensions doesn't matter. In fact, the tool will re-order in descending order regardless.

Descriptions are optional.
